# Creator Studio 제품 개발 청사진 v0.1

## 1. 제품 한 문장

> 화면을 녹화하는 순간부터 화면·카메라·마이크·아바타·커서가 각각 편집 가능한 프로젝트로 저장되는 로컬 우선 AI 크리에이터 스튜디오.

이 제품은 OBS 복제품도, 파이널컷 전체 복제품도 아니다. 핵심은 **촬영과 편집을 분리하지 않고, 촬영 시점부터 편집 가능한 원천 데이터와 이벤트를 남기는 것**이다.

## 2. 해결하려는 문제

기존 화면 녹화 도구는 대부분 촬영 당시의 레이아웃을 하나의 영상으로 굳혀 저장한다. 녹화 후 카메라 위치를 바꾸거나, 얼굴을 아바타로 교체하거나, 마우스 움직임을 기반으로 확대 구간을 다시 만들기 어렵다.

전문 편집기는 강력하지만 화면 녹화 제작자가 자주 사용하는 작업까지 지나치게 복잡하다. 반대로 간단한 녹화기는 편집 자유도가 낮다.

이 제품은 다음 사이의 틈을 노린다.

- OBS보다 촬영 설정이 쉽다.
- Loom보다 촬영 후 편집 자유도가 높다.
- CapCut보다 화면 녹화와 튜토리얼 제작에 특화된다.
- Final Cut/Premiere보다 배우기 쉽고 자동화가 강하다.
- VTube Studio류보다 녹화·편집·아바타 교체가 한 프로젝트 안에서 이어진다.

## 3. 핵심 사용자

### 3.1 튜토리얼·교육 제작자

- 소프트웨어 사용법, 코딩, SAP, 사내 교육 영상
- 화면과 설명자의 카메라를 함께 촬영
- 민감정보 블러, 커서 강조, 자동 확대가 중요

### 3.2 얼굴 공개를 원하지 않는 크리에이터

- 카메라 대신 2D/3D 캐릭터 사용
- 녹화 후 캐릭터·표정 강도를 교체하고 싶음

### 3.3 스트리머·버튜버

- 장면, 소스, 아바타, 오디오를 합성
- 방송보다 편집 영상 제작에 더 강한 도구가 필요

### 3.4 회사·컨설턴트

- 내부 화면과 자료를 외부 서버에 올릴 수 없음
- 자막과 노이즈 제거도 로컬 처리가 필요
- 프로젝트 복구와 데이터 보안이 중요

## 4. 제품 원칙

1. **원본 불변**: 편집은 원본 파일을 직접 수정하지 않는다.
2. **소스 분리**: 화면, 카메라, 마이크, 시스템 오디오, 아바타 모션을 별도로 저장한다.
3. **로컬 우선**: 기본 기능은 로그인이나 클라우드 없이 동작한다.
4. **촬영 실패 방지**: 화려한 효과보다 장시간 녹화와 충돌 복구를 먼저 완성한다.
5. **화면 제작 특화**: 일반 영화 편집 기능보다 커서·클릭·화면 확대·문장 편집을 우선한다.
6. **엔진 격리**: UI와 제품 도메인이 FFmpeg·MLT 내부 API에 직접 종속되지 않는다.
7. **플러그인 가능**: 캡처, 추적, 효과, AI 엔진을 교체할 수 있어야 한다.
8. **라이선스 우선**: GPL 코드를 실수로 폐쇄형 제품에 링크하지 않는다.

## 5. 제품 범위

### 5.1 Studio — 촬영

#### 소스

- 전체 모니터
- 특정 애플리케이션 창
- 지정 영역
- 웹캠·외장 카메라
- 마이크
- 시스템 오디오
- 이미지·영상·텍스트
- 브라우저 소스는 후순위
- 2D 아바타
- 3D VRM 아바타는 확장 단계

#### 장면

- 소스 조합을 장면으로 저장
- 장면 간 즉시 전환
- 장면 템플릿
- 프레젠테이션, 코딩, 강의, 게임, 쇼츠 프리셋

#### 녹화

- 화면, 카메라, 마이크, 시스템 오디오를 독립 트랙으로 저장
- 선택적으로 촬영 당시 합성 화면도 프리뷰 트랙으로 저장
- 일시정지·재개
- 마커 추가
- 단축키
- 디스크 잔량, 인코더 상태, 드롭 프레임 표시
- 강제 종료 후 복구

#### 인터랙션 데이터

- 커서 좌표
- 클릭 좌표·버튼
- 키보드 단축키 표시 이벤트
- 활성 창과 장면 변경
- 아바타 표정·포즈 파라미터
- 사용자 마커

키 입력은 비밀번호·민감정보를 저장하지 않도록 기본적으로 문자 자체가 아니라 `Ctrl+C`, `Shift`, `Enter` 등 표시 허용 키만 화이트리스트로 기록한다.

### 5.2 Editor — 편집

#### 필수 편집

- 멀티트랙 타임라인
- 컷, 분할, 트림, 리플 삭제
- 클립 이동, 복제, 잠금, 비활성화
- 스냅, 확대·축소, 프레임 단위 이동
- Undo/Redo
- 원본 재연결

#### 화면·카메라

- 위치, 크기, 회전, 자르기
- 둥근 모서리, 그림자, 테두리
- 크로마키와 배경 제거
- PIP 프리셋
- 키프레임

#### 오디오

- 독립 볼륨
- 음소거, 페이드
- 노이즈 제거
- 컴프레서, 리미터
- 라우드니스 정규화
- 파형 표시

#### 텍스트·자막

- 자동 자막
- 단어·문장 타임스탬프
- 자막 스타일 프리셋
- 화면 제목, 콜아웃, 화살표
- SRT/VTT 가져오기·내보내기

#### 화면 녹화 특화 편집

- 커서 자동 강조
- 클릭 리플·사운드
- 커서 위치 기반 자동 줌 제안
- 특정 앱·영역 자동 크롭
- 녹화 당시 카메라 레이아웃 재배치
- 화면·카메라·아바타를 16:9와 9:16에 재배치
- 민감정보 영역 블러 트래킹은 확장 단계

#### 텍스트 기반 편집

- 자막 문장을 삭제하면 해당 구간을 비파괴 편집으로 제거
- 침묵 구간 표시
- 반복어·필러 후보 표시
- 사용자 승인 후 리플 컷 적용

### 5.3 Avatar — 버튜버

#### 2D

- Inochi2D 모델 불러오기
- 얼굴 방향, 눈, 눈썹, 입, 표정 파라미터 매핑
- 투명 배경
- 추적 보정과 캘리브레이션
- 녹화 후 모션 재적용
- 캐릭터 교체

#### 3D

- VRM 1.0 불러오기
- 얼굴 BlendShape/Expression 매핑
- 상체 포즈
- 배경 투명화
- 2D 기능이 안정된 후 별도 모듈로 추가

#### 이번 제품에서 하지 않는 것

- 첫 버전에 전문 리깅 에디터를 만들지 않는다.
- Live2D Cubism 호환은 공식 SDK와 라이선스를 별도 검토한 후 확장한다.
- 모션캡처 스튜디오 수준의 전신 추적은 초기 범위가 아니다.

### 5.4 Export — 출력

- MP4 H.264 기본
- HEVC·AV1은 하드웨어와 라이선스 상태에 따라 제공
- 16:9, 9:16, 1:1
- YouTube 1080p/4K, Shorts, 강의 프리셋
- 자막 번인 또는 별도 파일
- 하드웨어 인코딩
- 렌더 큐, 일시정지, 실패 재개
- 투명 배경 영상 출력은 지원 코덱에서 확장

## 7. 제품의 결정적 차별 기능

### 7.1 Editable Capture

촬영 당시 합성 결과만 남기는 것이 아니라 원본 소스와 레이아웃 이벤트를 저장한다. 촬영을 끝낸 뒤에도 카메라를 옮기고 아바타를 교체하며 화면만 확대할 수 있다.

### 7.2 Cursor Intelligence

커서를 단순히 영상 픽셀로 녹화하지 않고 이벤트로 저장한다. 편집 단계에서 클릭 강조, 자동 줌, 커서 숨김, 부드러운 이동을 재생성한다.

### 7.3 Avatar Replay

아바타 자체를 영상으로만 저장하지 않고 얼굴 추적 파라미터도 저장한다. 편집 중 모델을 교체하거나 표정 강도를 낮출 수 있다.

### 7.4 Transcript Editing

문장을 지우면 영상 컷이 따라오는 편집 방식을 제공한다. 초보자는 타임라인을 몰라도 1차 편집을 끝낼 수 있다.

### 7.5 Privacy-first Local AI

화면 녹화에는 회사 시스템, 코드, 개인정보가 들어간다. 자막·노이즈 제거·기본 분석은 로컬 실행을 기본값으로 둔다.

## 8. 명확한 비범위

첫 상용 버전에서 다음을 목표로 삼지 않는다.

- Final Cut Pro의 모든 색보정·멀티캠·영화 후반작업 기능
- CapCut의 거대한 효과·음원·템플릿 마켓 전체
- OBS의 모든 스트리밍 프로토콜과 플러그인
- After Effects 수준 모션그래픽
- Blender 수준 3D 제작
- Live2D/Inochi2D 리깅 제작 도구 전체

이 제품이 이겨야 할 전장은 **화면 기반 콘텐츠의 촬영부터 빠른 편집까지**다.

## 9. 상용 버전 완료 기준

- Windows와 macOS에서 설치·업데이트 가능
- 1080p60 화면 + 1080p30 카메라 + 2개 오디오 트랙의 2시간 녹화 안정성
- 강제 종료 후 최근 세그먼트만 손실하고 프로젝트 복구
- 기본 컷 편집, PIP, 오디오, 자막, 출력
- 로컬 자동 자막
- 2D 아바타 추적·녹화·재편집
- 하드웨어 인코딩과 소프트웨어 폴백
- 프로젝트 자동 저장과 Undo/Redo
- 오픈소스 고지와 라이선스 구성 자동 검사
- 크래시 로그 내보내기와 장치 진단 리포트
