# R1-07 전체 검증 상태

검증 기준: Windows x64, 실제 화면·카메라·마이크·WASAPI 시스템 오디오 권한을 허용한 개발 PC.

## 통과한 항목

- 실제 30분 녹화 세션 생존.
- 화면·카메라·마이크·시스템 오디오 4개 트랙 생성.
- 세그먼트 저장과 세션 완료.
- 최대 동기 드리프트 약 0.29초 이하인 실행 확인.
- 시스템 오디오 장치 무효화 시 다른 소스 녹화 유지.
- 타임라인 프레임 경계가 한 프레임보다 짧은 양수 범위를 출력 프레임으로 양자화.
- 앱 188/188, 코어 485/485, FFmpeg 33/33, MLT 42/42 단위·통합 테스트.

## 보류된 항목

기본 2초 세그먼트로 30분을 녹화하면 약 3,600개 클립이 생성된다. 이 프로젝트를 MLT 그래프로 재오픈하는 과정에서 Windows 프로세스의 메모리와 핸들이 1.9GB/190만 수준까지 증가했고, 55분 관찰 후에도 편집기 재오픈·최종 MP4 export가 완료되지 않았다. 따라서 R1-07은 아직 완료로 표시하지 않는다.

재조립 단계의 중복 FFmpeg probe는 제거했지만, MLT 대량 클립 그래프 구성 자체의 확장성 개선이 남아 있다. 이 항목을 해결하고 동일한 30분 녹화→재오픈→편집→취소/재시도→MP4 probe 게이트를 다시 통과해야 R2를 시작한다.
